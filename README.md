# SCT_WD_4
To-Do-Web App

Project Overview:

This project is a basic To-Do web application developed using HTML, CSS, and JavaScript.
It enables users to efficiently manage daily tasks by allowing them to add, edit, organize, and track tasks with assigned date and time.

The application focuses on simplicity, usability, and clean UI design, making task management easy and effective.
